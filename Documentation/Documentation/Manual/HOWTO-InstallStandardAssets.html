How do I install or upgrade Standard Assets?

Desktop

Unity ships with Standard Assets and Pro Standard Assets. These are collections of assets that are widely used by most Unity customers. When you create a new project from the Project Wizard you can optionally include these asset collections. These assets are copied from the Unity install folder into your new project. This means that if you upgrade Unity to a new version you will not get the new version of these asset and so upgrading them is needed.

Standard Assets contain useful things like a first person controller, some skyboxes, flares, a water plane prefab and common camera scripts.

Pro Standard Assets contain all kinds of Image Effects, like Glow, Motion Blur, Color Correction, Noise and more, as well as several advanced Water prefabs.

Installing

When creating a new project Unity automatically installs the Standard Assets and Pro Standard Assets for Unity Pro users.

iOS

If you have an iOS license, then Unity will ship with Mobile Standard Assets which contain a first person controller, iOS specific shaders, projectors, shadows, common scripts and more.

Installing

When creating a new project Unity iOS automatically installs the Mobile Standard Assets.

Android

If you have an Android license, then Unity will ship with Mobile Standard Assets which contain a first person controller, Android specific shaders, projectors, shadows, common scripts and more.

Installing

When creating a new project Unity Android automatically installs the Mobile Standard Assets.


The Standard Assets are included in all new Projects by default

Upgrading

Sometimes you might want to upgrade your Standard Assets, for example because a new version of Unity ships with new Standard Assets:

  1. Open your project.
  2. Choose Assets->Import Package from the menu.
  3. Select Standard Assets.unitypackage or Pro Standard Assets.unitypackage or Mobile Standard Assets.unitypackage.
  4. A list of new or replaced assets will be presented, click Import.

Note: These packages can be found on disk at the following (default) locations:

  • PC: c:\program files\unity\editor\standard packages
  • Mac: /Applications/Unity/Standard Packages
