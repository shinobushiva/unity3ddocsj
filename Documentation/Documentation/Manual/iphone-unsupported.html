現在 Unity iOSでサポートされていない機能

グラフィックス

  • DXT テクスチャ圧縮はサポートされない; 代わりにPVRTC フォーマットを使う必要がある. Texture2D Component page にもっと情報があります.
  • Rectangular テクスチャはPVRTCフォーマットへ圧縮されない.
  • Movie Textures はサポートされない; フルスクリーンストリーミング再生を使う必要があります. Movie playback page にもっと情報がありますので見て下さい.
  • Terrain engine はサポートされない.
  • Tree Creator はサポートされない.
  • Open GL ES2.0 はiPhone, iPhone 3G, iPod Touch 1世代目 と iPod Touch 2世代目のハードウェアではサポートされない.
 

オーディオ

  • Ogg オーディオ圧縮はサポートされない. Ogg オーディオは、あなたがエディタをiOSプラットフォームに切り替えた瞬間自動的にMP3に変換される. AudioClip Component page にUnity iOSのオーディオサポートに関してもっと情報がありますので見て下さい.
 

スクリプト

  • OnMouseDown, OnMouseEnter, OnMouseOver, OnMouseExit, OnMouseDown, OnMouseUp, OnMouseDrag イベントはサポートされない.
  • Duck Typing のような動的機能(Dynamic Features)はサポートされない.  コンパイラが動的機能を強制的にエラーとして報告するように#pragma strictをあなたのスクリプトに使いなさい.
  •  via WWW class を介してのVideo streamingはサポートされない.
  • WWW class によるFTPサポートは制限されている.
 

Unity iOS Advanced Licenseで制限される機能

  • Static batching は Unity iOS Advancedだけでサポートされる.
  • Video playback iは Unity iOS Advancedだけでサポートされる.
  • Splash-screen customization は Unity iOS Advancedだけでサポートされる.
  • AssetBundles は Unity iOS Advancedだけでサポートされる.
  • Code stripping は Unity iOS Advancedだけでサポートされる.
  • .NET sockets は Unity iOS Advancedだけでサポートされる.

注記: 1 MB の .NET CIL コードは大ざっぱに 3-4 MB の ARM コードへと変換(translate)されるため、外部ライブラリへの参照を最小限にすることが推奨される . 例えば,もしstrippingが使われなければ、あなたのアプリケーションがSystem.dll と System.Xml.dllを参照するならば 6 MB のARM コードが追加される。いつかアプリケーションはリンカーがリンクする時に問題が起きる限界に達するでしょう. もしあなたがアプリケーションのサイズを大いに気にするのであれば、依存(依存関係)が少ないためにJavaScriptよりもC#が適している事を発見するでしょう.
