チュートリアル

チュートリアルではUnityを操作しながら使い方を勉強します。実際のプロジェクト開発を体験することができるでしょう。初めてのユーザは、GUIの基礎とスクリプトの基礎チュートリアルから始めると良いでしょう。その後は、どのチュートリアルでも大丈夫です。すべてPDFフォーマットで公開されていますので、印刷して読みながらUnityの操作を行えます。

Note: これらのチュートリアルはデスクトップバージョンのUnityについてです。AndroidやiOSデバイス (iPhone/iPad)では使用できません。

3DPlatformTutorial(日本語)

また、Unityのためのプレゼンテーション、記事、Assetsまたは拡張のような他のリソースを検索するなら、ここで見つけることができます。

また、弊社のUnity3Dチュートリアルのホームページをチェックして、チュートリアルについての最新の更新を確認することができます。
