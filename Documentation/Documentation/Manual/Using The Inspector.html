インスペクタの使い方

インスペクタはさまざまな種類のプロパティを表示、編集するために使用されます。

Unityのゲームはメッシュ、スクリプト、サウンド、またはライトのような他のグラフィック要素を含む複数のGameObectで構成されています。階層またはシーンビューでGameObjectを選択すると、インスペクタが表示され、GameObjectのすべてのコンポーネントやマテリアルのプロパティを変更できます。また、プロジェクトビューでプレハブを選択した場合、選択したものと同じものが生成されます。ゲーム内のGameObjectの機能を変更するときはこの方法で行います。 GameObject-Component relationshipを読んで理解することが非常に重要です。

 

インスペクタでGameObjectのコンポーネントとマテリアル等を表示しています。

カスタムコンポーネントとして動作するスクリプト（script）を自分で作成することができます。GameObjectにスクリプトコンポーネントが追加されている場合、そのスクリプトのメンバ変数はインスペクタで直接編集可能なプロパティ(Properties)として表示されます。これにより、スクリプト自体を変更することなく、スクリプト変数を変更することができます。

Furthermore, the Inspector is used for showing import options of assets such as textures, 3D models, and fonts when selected. Some scene and project-wide settings are also viewed in the Inspector, such as all the Settings Managers.

さらに、インスペクタはフォント、3Dﾓﾃﾞﾙ､ﾃｸｽﾁｬｰなどの選択されたasset のオプションを表示するのに使われます。いくつかのシーン、プロジェクト全体の設定はすべてSettings Managersなどインスペクタで表示されます。

 

Any property that is displayed in the Inspector can be directly modified. There are two main types of Properties: Values and References.

インスペクタに表示される任意のプロパティを直接変更することができます。プロパティの主な機能として値（Values）と参照（References）の2つがあります。
