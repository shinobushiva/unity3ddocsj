3Dアプリケーションからどのようにオブジェクトをインポートすればいいですか？

 

Unityは、最も人気のある3Dアプリケーションからのインポートをサポートしています。ご利用のアプリケーションを選択してください：

 
  • Maya
  • Cinema 4D
  • 3ds Max
  • Cheetah3D
  • Modo
  • Lightwave
  • Blender
 

その他のアプリケーション

Unityでは以下の can read .FBX, .dae, .3DS, .dxf and .obj ファイルを読み込むことができますので、もしご利用のアプリケーションがこれらのフォーマットでファイルを書き出すことができる場合ご使用になることができます。FBX exportersがオートデスクからでていますので参考にしてください。日本語はこちらから、 英語はこちらから。いくつかのアプリケーションではCollada exporteも利用可能です。

 

 

ヒント

  • Store textures in a folder called Textures next to the exported mesh.Textures という名前のフォルダーの中のテクスチャThis will guarantee that Unity will always be able to find the Texture and automatically connect the Texture to the Material. 詳細については、テクスチャのリファレンスを参照してください。For more information, see the Textures reference.
