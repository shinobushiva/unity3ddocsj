static function InitializeServer (connections : int, listenPort : int, useNat : bool) : NetworkConnectionError

説明

サーバーを初期化します。

connectionsは、接続が許可された着信接続の数、またはゲームで許可されているプレイヤーの数です。IistenPortはポート番号です。useNatは、NATパンチスルー機能を設定します。このサーバーは、ファシリテーターを用い、NATパンチスルーを使用した接続を受け入れる場合、これをtrueに設定します。
