タグとはなんですか？(What is a Tag?)

タグ(Tag)は、複数のゲームオブジェクト(GameObject)にリンクする事のできる、言葉です。例えば、"Player"や"Enemy"というタグを定義して、プレイヤがコントロールするキャラクタや、プレイヤ以外のキャラクタに使えます。また、"Collectable"と言うタグはプレイヤがシーン(Scene)中で拾うことの出来るアイテムに使えます。タグはスクリプトからゲームオブジェクトを特定するために使用します。スクリプトコードに置いて、特定のタグを持ったゲームオブジェクトを発見することが可能です。これはGameObject.FindWithTag()関数によって行えます。

例えば:

				// Instantiates respawnPrefab at the location 
// of the game object with tag "Respawn"

var respawnPrefab : GameObject;
var respawn = GameObject.FindWithTag ("Respawn");
Instantiate (respawnPrefab, respawn.position, respawn.rotation);

これによって、ゲームオブジェクトをプロパティに手動でドラッグアンドドロップして追加する手間を省けます。同じスクリプトを複数のゲームオブジェクトで使用する場合は、とても効果的です。もうひとつの例としてトリガーコライダー(Trigger Collider) のコントロールスクリプトが、プレイヤーや敵と作用する場合があります。同様に、拾えるアイテムにも使えます。タグはこの種のテスト(衝突検出など)を簡単にします。

タグの適用(Applying a Tag)

インスペクタ(Inspector)に、タグとレイヤー(Layer) のドロップダウンメニューが表示されます。ゲームオブジェクトの名前のすぐ下です。タグを適用するには、タグのドロップダウンを開いて、使いたいタグを選択します：

ゲームオブジェクトがタグに関連付けられました。

新しいタグの作成(Creating new Tags)

新しいタグを作るには、ドロップダウンメニューの一番下の"Add new Tag..."オプションを選択します。選択するとタグマネージャー(Tag Manager)がインスペクタに表示されます。タグマネージャーはここを参照してください。

レイヤーもタグと同じように表示されます。しかし、通常はUnityがゲームオブジェクトを描画する方法を指定するために利用します。詳しくはレイヤー(Layers)のページを参照してください。

ヒント

  • ゲームオブジェクトはタグを１つだけ割り当てることが出来ます。
  • Unityはいくつかのビルトインタグを提供しており、これらはタグマネージャーには表示されません。
    • "Untagged"
    • "Respawn"
    • "Finish"
    • "EditorOnly"
    • "MainCamera"
    • "Player"
    • "GameController".
  • タグには好きな言葉を使えます。(短い文章を使うことも出来ますが、インスペクタの幅を広げないと内容が確認できません)
