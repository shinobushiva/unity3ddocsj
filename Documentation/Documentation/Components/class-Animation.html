アニメーション(Animation)


アニメーションのインスペクター
 

Properties

スクリプトからアクセスすることができるアニメーションの一覧です。
   
Animation デフォルトアニメーションはPlay Automaticallyが有効の時再生されるでしょう。
Animations スクリプトからアクセスできるアニメーションのリストです。
Play Automatically ゲームを始めるときにアニメーションを自動的に再生するべきですか？
Animate Physics アニメーションは物理的作用をするべきです。
Animate Only If Visible 有効の時、オブジェクトが画面上に表示されていない場合、現在再生中のアニメーションが再生されません。

Unity側でどのようにアニメーションを作成するかの情報は Animation View Guide を見て下さい。 キャラクターのアニメーションをインポートする方法は Animation Import を、またはゲームのためにアニメーションの行動を作成する方法は Animation Scripting のページを見て下さい。
