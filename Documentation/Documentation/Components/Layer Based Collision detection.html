レイヤーに基づいた衝突判定

Unity 3.xではレイヤーに基づいた衝突判定が含まれています。これはゲームオブジェクトが特定のレイヤに配置されたゲームオブジェクトとだけ衝突することを可能にします。


同一レイヤーのオブジェクトと衝突

上の画像では、６つのゲームオブジェクト(３つの平面と、３つの立方体)が表示されています。右側の"Collision Matrix(衝突マトリックス)"はどのオブジェクトがどのレイヤーと衝突するかを表しています。例えば、衝突マトリックスを同一のレイヤーに属するオブジェクトの間でだけ衝突が起こるように設定することが出来ます。

ゲームオブジェクトにレイヤーに基づいた衝突検出を設定する

  1. ゲームオブジェクトが諸属するレイヤーを選択します。
  2. １をそれぞれのゲームオブジェクトについて繰り返します。
  3. Physics Preference Panel(物理設定パネル)をEdit->Project Settings->Physicsから開きます。
  4. 衝突マトリックス上で衝突を検出する対象のレイヤーをチェックします。
