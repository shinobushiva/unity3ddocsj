キューブマップテクスチャ(Cubemap Texture)

キューブマップテクスチャは架空のキューブで表現して知らせる６つの面に分けられたテクスチャのコレクションです。ほとんどの場合オブジェクト上の無限遠方の反射を表示するのに使い、バックグラウンド上の遠方風景をどのようにスカイボックス(Skybox)で表示するかに似ています。Unity上のビルトインシェーダの反射(Reflective)は反射表示のためにキューブマップを使います。


山のシーンのキューブマップはこのスフィア上の反射のように表示されます

あなたはいくつかの方法からキューブマップを作成できます:

  1. Assets->Create->Cubemapを使い、 プロパティをセットし、６つのテクスチャ(Texture)アセットをキューブマップの"faces"に対応した場所にドラッグします。
  2. 単一のインポートテクスチャアセットからキューブマップを作成するためにテクスチャ(Texture)インポートセッティングを使用してください。
  3. スクリプトからキューブマップにシーンをレンダリングします。Camera.RenderToCubemap上のコード例は、エディタから直接キューブマップをレンダリングするためのスクリプトが含まれています。

Properties

   
Right (+X) キューブマップ表面の右(right)のグローバルサイドのためのテクスチャです。
Left (-X) キューブマップ表面の上(up)のグローバルサイドのためのテクスチャです。
Top (+Y) キューブマップ表面の最上段(top)のグローバルサイドのためのテクスチャです。
Bottom (-Y) キューブマップ表面の下(bottom)のグローバルサイドのためのテクスチャです。
Front (+Z) キューブマップ表面の前(forward)のグローバルサイドのためのテクスチャです。
Back (-Z) キューブマップ表面の後ろ(rear)のグローバルサイドのためのテクスチャです。
Face Size それぞれの個々のキューブマップ表面の向こう側の画素の幅と高さです。テクスチャは内部的にこのサイズに合うようスケールされる、アセットを手動でスケールする必要はありません。
Mipmap ミニマップを作成するために有効にします。
Format 作成されたキューブマップのフォーマットです。
