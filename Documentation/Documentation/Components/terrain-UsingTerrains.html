地形の使い方(Using Terrains)

 デスクトップ(Desktop)

新しい地形の作成

新しい地形(Terrain)はTerrain->Create Terrainから、作成できます。地形がプロジェクトと 階層ビューに追加されます。


新しい地形はシーンビュー内で、次のように表示されます：
シーンビュー内の新しい地形

もし、地形のサイズを変更したければTerrain->Set Resolutionをメニューバーから選択してください。いくつかの地形に関する設定が、ダイアログで行えます。


地形の解像度を設定

上の画像は、変更できる値です。
それぞれ:

  • Terrain Width: 地形の幅です。(ユニット単位)
  • Terrain Height: 地形の高さです。(ユニット単位)
  • Terrain Length: 地形の長さです。(ユニット単位)
  • HeightMap Resolution: 選択された地形の高さマップ解像度です。
  • Detail Resolution: 草や、詳細メッシュをコントロールするマップの解像度です。パフォーマンスの面(描画回数を減らす)から、低い値の方が良いでしょう。
  • Control Texture Resolution: 地形に異なるテクスチャをレイヤして描くのに用いる、スプラットマップの解像度です。
  • Base Texture Resolution: 特定の距離のスプラットマップに使用される、合成されたテクスチャの解像度です。

地形の操作(Navigating the Terrain)

地形は他のゲームオブジェクトと少し異なる働きをします。最も違う点は、地形の位置を決めるためのトランスフォームギズモが無いことです。代わりに、ブラシ(Brushes)を使い、地形を操作します。地形の位置を変えたければ、Transform Positionの値をインスペクタで変更します。これで、地形を動かすことはできますが、回転したり、サイズを変更したりはできません。

階層において、地形が選択されている状態でF(focus)キーにより、優雅に地形を動かせます。Fキーを抑えたまま、マウスを移動すると、シーンビューのセンターが移動します。これにより、あるエリアを操作し、素早く異なるエリアにズームし、何かを変更することができます。もし、全ての地形が、シーンビューの中央に表示されていると、Fキーを押してもマウスが地形のエリアを移動しません。

地形の編集

地形が選択された状態でインスペクタを確認してください。新しい地形編集ツールが並んでいます。


インスペクタに表示された地形編集ツール

それぞれの長方形のボタンは異なる地形ツールです。高さを変更するもの、スプラットマップを塗るもの、木や岩などの付属物を追加するものがあります。クリックすると、それぞれのツールが使えます。それぞれのツールに関する、簡単な説明がボタンの下に表示されます。

ほとんどのツールはブラシを使います。ツールを選ぶと多くの種類のブラシが表示されます。ブラシを選択するには、クリックしてください。選択されたブラシは、指定した大きさに合わせて、地形の上にマウスを移動すると、プレビューを表示します。

ブラシはシーンビューで、直接、地形を塗るのに使用します。 ツールとブラシを選択し、地形上でクリック＆ドラッグすると、即座に変更されます。高さ、テクスチャ、装飾物を塗るには、階層ビューで地形を選択しておかなければなりません。

Note: ブラシが選択された状態で、マウスをシーンビュー内の地形上で動かし、Fキーを押します。マウスポインタの位置がシーンビューで中心に表示されブラシの大きさ(Brush Size)に合わせて、自動的にズームします。これが、編集中に、最も簡単で素早く、地形を見回す方法です。

地形キーボードショートカット(Terrain Keyboard Shortcuts)

地形インスペクタがアクティブな状態で、次のキーボードショートカット(Unityの設定で変更できます)を使うと、素早く編集を行えます：

  • Shift-Q から Shift-Y で地形ツールを選択します。
  • カンマ(,)とドット(.)でアクティブなブラシを選択します。
  • Shift-カンマ (<) and Shift-ドット (>)で、アクティブな 木/テクスチャ/詳細 (tree/texture/detail) オブジェクトを切り替えます。
