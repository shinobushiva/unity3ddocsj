高さ(Height)

 デスクトップ(Desktop)

地形編集ツールの使い方はシンプルです。文字通り、地形を Using any of the Terrain editing tools is very simple. You will literally paint the Terrain from within the シーンビューにおいて、塗りつぶします。高さツール(height tools)や他の全てのツールは、選択し、シーンビューにおいて、地形をクリックし、リアルタイムに編集を行います。

高さの引き上げと押し下げ(Raising & Lowering)

左側の１番目のツールは高さの引き上げ(Raise Height)ツールです。
.

このツールで、地形を塗ると、その部分の高さが引き上げられます。一回のクリックで、高さが増加します。マウスボタンを押したまま移動すると、最大値になるまで、連続的に高さが増加します。

いろいろなブラシを使って異なる効果を得られます。

高さを押し下げたい場合は、Shiftキーを押しながら操作します。

Note: ブラシが選択された状態で、マウスをシーンビュー内の地形上で動かし、Fキーを押します。マウスポインタの位置がシーンビューで中心に表示されブラシの大きさ(Brush Size)に合わせて、自動的にズームします。これが、編集中に、最も簡単で素早く、地形を見回す方法です。

高さを塗る(Paint Height)

左から２つ目のツールは高さを塗る(Paint Height)ツールです。
.

このツールは、高さを指定し、地形の一部を、その高さまで引き上げます。地形が指定した高さになると、高さの変更を停止します。

高さを指定するには、Shiftキーを押しながら、地形の、指定したい高さをクリックします。または、インスペクタ上でHeightスライダを調整します。

高さを指定すると、クリックするごとに、地形が引き上げられたり押し下げられたりします。

高さの平滑化(Smoothing Height)

左から３つめのツールは高さの平滑化ツールです。
  .

このツールは、塗っているエリア中の高さの差を滑らかにします。他のブラシと同様、シーンビューにおいて、滑らかにしたい部分を塗ります。

ハイトマップを使う(Working with Heightmaps)

Photoshopや実世界の地形データから作成した、グレースケールのハイトマップを読み込んで、地形にすることができます。Terrain->Import Heightmap - Raw...を選択し、使用するRAW ファイルを選択します。インポートの設定が表示されます。値はセットされていますが、地形のサイズをダイアログで変更することができます。準備ができたら、Importボタンをクリックします。 ハイとマップが地形に適応されたら、通常通り、ツールを使って編集できます。UnityのハイトマップインポータはグレースケールのRAWファイルのみをインポートできます。RGBチャンネルを用いてハイトマップを作ることはできません。

Unityは16ビットのフル解像度で作られたRAWファイルを扱えます。 Unity works with RAW files which make use of full 16-bit resolution. Any other heightmap editing application like BryceやTerragen、Photoshop等のハイトマップ編集アプリケーションで作ったフル解像度のハイトマップが使えます。

また、ハイトマップをRAWフォーマットで書きだすこともできます。Terrain->Import Heightmap - Raw...を選択すると、エクスポート設定ダイアログが表示されます。変更してExportをクリックすると、ハイトマップが保存されます。

Unityは地形を平坦にする簡単な方法を提供します。Terrain->Flatten...を選択すると、ウィザードで設定した高さで、地形を平坦にします。
