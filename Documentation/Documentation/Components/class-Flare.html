フレア(Flare)

フレアオブジェクトはレンズフレアコンポーネント(Lens Flare Components)によって使用されるソースアセットです。フレア自体はテクスチャファイルの組み合わせと、フレアの動作方法を決定する特定の情報です。あなたがシーン内でフレアを使いたい時、あなたはゲームオブジェクトにレンズフレアコンポーネントが付加した特定のフレアを参照します。

スタンダードアセット(Standard Assets)パッケージの中にいくつかサンプルフレアがあります。あなたのシーンにいずれかを追加したい場合、ゲームオブジェクトにレンズフレア(Lens Flare)コンポーネントを付加し、ちょうどメッシュレンダラにマテリアルを割り当てるように、レンズフレアのフレアプロパティに使用したいフレアをドラッグします。


フレアインスペクター

フレアは単一のテクスチャ上でいくつかのフレアエレメンツを含むことで働きます。フレアの中では、あなたがテクスチャのいずれかから含むようにしたいエレメンツを選択します。

プロパティ(Properties)

   
Elements フレア画像の数がフレアに含まれています。
Image Index エレメンツのためにフレアテクスチャからどのフレアイメージを使用するか。詳細については以下のフレアテクスチャ(Flare Textures)を参照してください。
Position エレメンツは、収容しているゲームオブジェクトの位置からスクリーンのセンターまで実行しながら、ラインに沿って埋め合わせします。0 = GameObject position, 1 = screen center.
Size エレメンツのサイズです。
Color エレメンツのカラーの色合いです。
Use Light Color フレアはライトのために付加するなら、ライトのカラーとフレアに色合いをつけることを可能にします。
Rotate 有効の場合、エレメンツの下部はスクリーンのセンターに常に表示され、レンズフレアがスクリーン上を移動するようにエレメンツはスピンしてます。
Zoom 有効の場合、エレメンツは目に見える（visible）ようになる時スケールアップし、そうでないとき再びスケールダウンします。
Fade 有効の場合、エレメンツは目に見える（visible）ようになる時限界までフェードインし、そうでないときフェードアウトします。
Flare Texture テクスチャはフレアのエレメンツによって使われるイメージを含みます。テクスチャレイアウトオプションのひとつに応じて配置しなければなりません。
Texture Layout どのように個々のフレアエレメンツ画像がフレアテクスチャの内側に配置されるかです。
Use Fog 有効の場合、フレアはフォグの間隔で消えていきます。これは小さいフレアのために一般的に使用されます。

詳細(Details)

フレアはラインに沿って配置された複数のエレメンツで構成されています。ラインはスクリーン中央のフレアレンズを含むゲームオブジェクトの位置を比較することで計算されています。ラインはゲームオブジェクトとスクリーン中央を超えて伸びます。すべてのフレアエレメンツはこのライン上に張られます。

フレアテクスチャ(Flare Textures)

パフォーマンス上の理由から、ひとつのフレアのすべてのエレメンツは同じテクスチャを共有する必要があります。このテクスチャは、単一のフレア上でエレメンツとして利用可能で様々なイメージのコレクションを含みます。どのようにエレメンツがフレアテクスチャに配置されるかテクスチャレイアウトが定義します。

テクスチャレイアウト(Texture Layouts)

別のフレアのテクスチャレイアウトを持っているオプションです。画像内の数字はそれぞれのエレメンツのためにイメージインデックスプロパティに相当します。

   
1 Large 4 Small
あなたが他より高い忠実性を持つために、エレメンツの１つを必要とする大きいサンスタイル（sun-style）フレアのためにデザインされます。これは広さより２倍高いテクスチャとして使われるようデザインされています。
1 Large 2 Medium 8 Small
１つの高明細、２つの中明細と８つの小さな画像を必要とする複雑なフレアのためにデザインされています。これは２つのミディアムエレメンツは虹色のサークルで、スタンダードアセットの"50mm Zoom Flare"で使用されています。これは広さより２倍高いテクスチャとして使われるようデザインされています。
1 Texture
シングルイメージ。
2x2 grid
シンプルな2x2グリッド。
3x3 grid
シンプルな3x3グリッド。
4x4 grid
シンプルな4x4グリッド。

ヒント(Hints)

  • あなたが多くの異なったフレアを使う場合、すべてのエレメンツを含む単一のフレアテクスチャを使うとベストなレンダリングパフォーマンスを得るでしょう。
  • レンズフレアはコライダによってブロックされます。たとえコライダがメッシュレンダラを持っていないとしても、フレアゲームオブジェクトとカメラはコライダの中間に隠します。
