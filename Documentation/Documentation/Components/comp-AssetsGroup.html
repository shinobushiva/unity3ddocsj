アセットコンポーネント(Asset Components)

アセットはあなたのゲームを作るモデル、テクスチャ、サウンドと他すべての"コンテンツ"ファイルです。

このセクションではアセットのすべての種類のコンポーネントを説明します。アセットの全体的な概要については Assets overview page を見て下さい。

  • Audio Clip
  • Cubemap Texture
  • Flare
  • Font
  • Material
  • Meshes
  • Movie Texture
  • Render Texture
  • Text Asset
  • Texture 2D
