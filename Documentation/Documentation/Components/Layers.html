レイヤー(Layers)

レイヤーはカメラがシーンの一部をレンダリングする場合や、ライトがシーンの特定の物体のみを照らすために、よく使われます。しかし、衝突を判定するために選択的にコライダーを無視するようなレイキャストにも使用できます。

レイヤの作成

始めのステップはGameObjectに割り当てられるレイヤを作成することです。新しいレイヤーは、エディットメニューを開きProject Setting->Tagsを選択します。

User Layerの開いている場所に新しいレイヤーを作りました。ここでは、８番のレイヤー(User Layer 8)を選択しました。

レイヤーの割り当て

新しいレイヤーを作成したら、レイヤーをゲームオブジェクトに割り当てなければなりません。

タグマネージャで８番レイヤのPlayerを割り当てます。

カメラのカリングマスク(Culling Mask)を用いてシーンの一部だけを描画する

カメラのカリングマスクを使用すると、特定のレイヤーに属するオブジェクトを選択的にレンダリング出来ます。まず、選択的にレンダリングを行うカメラを選択します。

レイヤーのチェックを操作して、カリングマスク属性を編集します。

レイを選択的にキャスト

レイヤを使用すると、特定のレイヤに含まれるコライダを無視したレイキャストが行えます。例えば、Playerレイヤのみにレイキャストして、他のもの全てを無視したい場合があるでしょう。

Physics.Raycast関数は、特定のレイヤを無視するかどうかを表すビットマスクを引き数として受け取ります。 layerMaskの全てのビットがオンであれば全てのコライダーと衝突します。layerMask = 0の時、レイキャストがコライダーを検出することはありません。

			// bit shift the index of the layer to get a bit mask
var layerMask = 1 << 8;
// Does the ray intersect any objects which are in the player layer.
if (Physics.Raycast (transform.position, Vector3.forward, Mathf.Infinity, layerMask))
    print ("The ray hit the player");

実際は、反対のことをしたい場合があります。レイをPlayer以外の全てのコライダーに対して衝突するようにしたい場合です。

			function Update () {
  // Bit shift the index of the layer (8) to get a bit mask
  var layerMask = 1 << 8;
  // This would cast rays only against colliders in layer 8.
  // But instead we want to collide against everything except layer 8. The ~ operator does this, it inverts a bitmask.
  layerMask = ~layerMask;

  var hit : RaycastHit;
  // Does the ray intersect any objects excluding the player layer
  if (Physics.Raycast (transform.position, transform.TransformDirection (Vector3.forward), hit, Mathf.Infinity, layerMask)) {
    Debug.DrawRay (transform.position, transform.TransformDirection (Vector3.forward) * hit.distance, Color.yellow);
    print ("Did Hit");
  } else {
    Debug.DrawRay (transform.position, transform.TransformDirection (Vector3.forward) *1000, Color.white);
    print ("Did not Hit");
  }
}

Raycast関数にlayerMaskを渡さない場合、IgnoreRaycastレイヤーを使用するコライダーのみを無視します。これは、レイキャスト時にコライダーを無視する最も簡単な方法です。
